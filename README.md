## 脚本
```bash
mvn flyway:migrate
```
也可以手动执行resources/db/migration下的sql文件  
```bash
mvn '-Dmybatis.generator.overwrite=true' mybatis-generator:generate
```

## 如何运行
1.安装jdk，maven，idea等  
2.克隆代码到本地  
3.依次运行上述两个脚本  
4.在idea中运行
